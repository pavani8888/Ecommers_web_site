# 🛒 Shopping Cart Application

A responsive Shopping Cart web application built using **HTML, CSS, and JavaScript**.

## 📌 Features

* Display products dynamically
* Search products instantly
* Add products to cart
* Remove products from cart
* Update product quantity
* Calculate total price automatically
* Store cart data using Local Storage
* Responsive design for:

  * Desktop 💻
  * Tablet 📱
  * Mobile 📱
* Modern UI with hover effects and animations

## ⚙️ How to Run

1. Download or clone the project.
2. Place all product images in the project folder.
3. Open `index.html` in your browser.

No installation or server setup is required.

---

## 🛍️ Functionalities

### Product Search

Users can search products using the search bar.

### Add to Cart

Products can be added to the cart using the **Add to Cart** button.

### Quantity Update

Users can increase or decrease item quantities.

### Remove Product

Products can be removed from the cart.

### Cart Persistence

Cart items are stored in browser local storage and remain available after page refresh.

## 📱 Responsive Design

### Desktop

### Tablet

* 2 product cards per row
